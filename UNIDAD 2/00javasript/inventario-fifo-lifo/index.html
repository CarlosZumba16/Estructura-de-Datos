Inventario FIFO vs LIFO — Tienda Tecnológica

Simulador didáctico para comparar ambos métodos de gestión de inventario

¿Cómo usar este simulador?
1. Elige un producto, cuántas unidades compras y a qué precio de costo.
2. Pulsa "Agregar lote" — el producto entra a la bodega de ambos métodos al mismo tiempo.
3. Ingresa cuántas unidades quieres vender y pulsa "Vender".
   → FIFO despacha primero el lote más antiguo (el que llegó antes).
   → LIFO despacha primero el lote más reciente (el que llegó después).
4. Observa cómo el costo y la ganancia cambian según qué lote se vende primero.
